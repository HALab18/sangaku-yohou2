座標を指定して予報を取得

地図で調べた緯度・経度を貼り付けると、その地点の山岳気象予報が見られます

山名で見つからない場所（登山道の途中・幕営地・沢の出合など）でも、緯度・経度さえ分かれば 山名検索やGPS現在地と同じ精度の予報を表示できます。

緯度・経度の調べ方（国土地理院「地理院地図」を使う場合）
  1. 地理院地図を開く ▸
  2. 知りたい地点が画面中央の十字（＋）に来るよう、地図をドラッグ・ズームして合わせる
  3. 画面上部の「共有」ボタンを押すと、URL欄に緯度・経度を含むリンクが表示されるのでコピーする
  4. コピーしたURL（または緯度・経度の数値だけ）を下の欄にそのまま貼り付ける
